Data Quality

Data Quality Audit Toolkit

用 Python + SQL 對 5 張電商資料表跑 9 類資料品質檢核（缺值、重複、格式、異常值、日期邏輯、參照完整性、金額計算），並計算 Completeness / Uniqueness / Validity / Consistency 四維度分數。

本次跑出：2,164 筆記錄、20 種問題類型、整體品質分數 97.3/100。
